# Youtube Downloader
this is a test version of `downloader`

### Get The package

```shell
# clone the repositorie
git clone https://github.com/Abderrahman-byte/yt-downloader.git
cd yt-downloader
# if you are in Linux
sudo chmod +x install.sh
# installation
./install.sh
```

### Usage

```shell
downloader <video-link> -l <input-dir>
```
